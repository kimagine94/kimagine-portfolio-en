Image Analysis Using Tensorflow

텐서플로우를 이용한 이미지 분석

Using a single layer of DNN, we can model 5 different types of information from 1,031 image data (facial photographs) including indoor / outdoor, weekend / weekday, breakfast / lunch / dinner, couple / solo .

  • Research period : 2016.12 - 2017.07
  • Related Technology : Python, Tensorfolw, Classification, Deap Learning, DNN

Copyrighted Designs Classification Based on CNN

CNN기반의 저작권이 있는 디자인 분류

In VGGNet, we have implemented a model that can classify 20,000 pieces of image data collected through Instagrams into 5 pieces of copyright information by using CNN with several layers.

  • Research period : 2017.07 - 2017.12
  • Related Technology : Python, Tensorfolw, Classification, Deap Learning, CNN
